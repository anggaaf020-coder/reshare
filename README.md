# ReShare

ReShare adalah aplikasi web berbasis PHP native yang bertujuan untuk memfasilitasi kegiatan donasi dan pengambilan barang bekas layak pakai. Aplikasi ini dirancang untuk mengurangi limbah rumah tangga sekaligus membantu masyarakat yang membutuhkan melalui sistem digital yang sederhana, modern, dan mudah digunakan.

ReShare mengusung konsep eco-friendly platform dengan tampilan minimalis dan fokus pada kemudahan distribusi barang.

## Fitur Utama
- Autentikasi pengguna (Login & Register)
- Login menggunakan Google OAuth
- Manajemen akun pengguna (profil, ubah username)
- Upload barang donasi lengkap dengan foto dan deskripsi
- Katalog barang donasi
- Manajemen event donasi
- Sistem database terintegrasi (MySQL)
- Desain UI modern menggunakan Tailwind CSS
- Struktur kode terorganisir (frontend & backend terpisah)

---

## Cara Menjalankan Website (Setup)
- Menjalankan apache dan mysql di xampp
- membuka localhost yang merujuk ke page index localhost/reshare/frontend/index.php
